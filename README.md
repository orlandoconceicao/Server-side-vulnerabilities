## 🎯 Objetivo

Aprender na prática como identificar, explorar e entender vulnerabilidades server-side como:

- Information Disclosure  
- Path Traversal  
- File Read Vulnerabilities  
- Manipulação de requisições HTTP  
- Falhas comuns em aplicações web  

---

## 🧠 O que estou aprendendo

- Como aplicações web processam requisições HTTP  
- Como identificar parâmetros vulneráveis  
- Uso do Burp Suite (Proxy, HTTP History, Repeater)  
- Análise de respostas do servidor  
- Pensamento de atacante (Recon + Exploração)  

---

## 🛠️ Ferramentas utilizadas

- Burp Suite  
- Browser (Chrome / Burp Browser)  
- DevTools (F12)  
- Web Security Academy labs  

---

## 🚀 Fluxo de estudo

```
Abrir lab
↓
Intercept / HTTP History
↓
Enviar para Repeater
↓
Modificar requisição
↓
Analisar resposta
↓
Entender vulnerabilidade
```

---

## 💬 Aprendizado principal

> Segurança web não é sobre “hackear”, e sim entender como a aplicação funciona e onde ela confia demais no usuário.
